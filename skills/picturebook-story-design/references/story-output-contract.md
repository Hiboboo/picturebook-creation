# 绘本故事输出契约

输出必须使用中文，并严格按以下结构交付。不能把“构图说明”写成“已生成图片”；若当前无法出图，必须明确标记为“未生成”。

## 1. 交付状态

- 当前状态：故事草案 / 视觉设定 / 样张确认 / 全书出图 / 排版完成 / 质检完成
- 实际图片生成：是 / 否
- 生成方式：人工 / AI 辅助 / AI 生成后人工修整 / 其他
- 本次未完成项：
- 需要用户确认项：

## 2. 基本信息

- 系列：
- 书名：
- 主题：
- 图/文：小葫芦
- 主要适合年龄：2-3 岁 / 3-4 岁 / 4-6 岁
- 可延伸年龄：
- 核心主题：只写一个。
- 孩子的生活入口：孩子熟悉的具体情境或经验。
- 目标情绪变化：例如“从犹豫，到感到有人陪，再到愿意做一个小动作”。
- 故事页数：不含封面。
- 画面风格：媒介、线条、颗粒、人物比例和总体气质。
- 视觉细节密度：简洁 / 适中 / 较丰富；默认适中偏简洁。
- 色彩设计策略：主色、辅助色、强调色及其情绪功能。
- 图片比例：固定 `3:4` 竖版。
- 锁定像素尺寸：例如 `1536×2048` 或 `1800×2400`；整本不得混用。
- 文字安全边距：默认至少画布宽高的 `6%`；印刷项目另列出血。
- 固定输出目录：`outputs/picturebooks/<series-slug>/<title-slug>/`

## 3. 儿童发展与情绪安全策略

- 孩子可能感受到的情绪：
- 故事如何接纳这种情绪：
- 孩子拥有的选择：观察 / 牵手 / 抱安抚物 / 分步骤 / 暂停 / 换一种表达 / 下次再试；按故事实际填写。
- 成人如何提供支持：写具体动作和语言，不写空泛“鼓励”。
- 故事中的最小可行尝试：孩子无需完成全部任务也能获得满足感的小步骤。
- 结尾如何避免“必须克服”：
- 奖励、掌声或贴纸的处理：如出现，说明其不是价值判断或条件式奖励。
- 禁用表达：列出本故事不能出现的催促、比较、羞辱或成人化语言。

## 4. 故事设定卡

- 背景：
- 儿童视角：说明故事来自孩子能理解的生活经验或身体感受。
- 故事弧线：生活入口 → 愿望/好奇 → 犹豫信号 → 安全支架 → 一个小动作 → 真实回应 → 温暖收束。
- 重复结构：重复句式、动作、问题、拟声词或韵律；说明每次如何变化。
- 页间推进：概述每页新增的动作、发现、关系或情绪变化。
- 温暖结尾：说明如何让孩子感到被理解、安心和有余韵，而不是直接说教。
- 故事禁改项：核心主题、关键情绪逻辑、结尾边界等。

## 6. 全书镜头与翻页计划

先给出一张全书总表，每页一行：

| 页码 | 叙事节点 | 情绪起点→终点 | 景别 | 视角/机位 | 出场角色 ID | 核心动作 | 空间锚点 | 文字区 | 相比上一页的视觉新增量 |
|---|---|---|---|---|---|---|---|---|---|

必须满足：

- 8 页及以上至少有 3 种景别和 2 种合理视角。
- 相同机位、相同景别和相近站位通常不连续超过 2 页。
- 至少包含建立场景、情绪近景或动作特写、关键转折和满足性结尾画面。
- 镜头变化服务故事，不破坏空间轴线。

## 7. 封面图片

必须给出：

- 封面状态：已生成 / 未生成 / 需重做。
- 无字封面图片文件路径：
- 排版完成封面图片路径：
- 图片比例与像素尺寸：固定 `3:4`，与内页一致。
- 封面中心意思：一眼呈现主角、核心情境和故事气质，但不剧透全部结局。
- 封面构图说明：主体、背景、视线、空间关系、留白和视觉层级。
- 角色锁定：列出出场角色 ID、服装和道具。
- 书名排版：大字号、第一视觉层级、普通清楚、最多 2-3 行。
- 署名排版：固定 `图/文：小葫芦`；普通字体、黑色或深灰、小字号、舒适间距。
- 标题安全区：不得贴边或遮挡主角脸部和关键动作。
- 无字封面图像提示词：不得要求模型生成任何文字。
- 封面负面提示词：包含安全、解剖、一致性、乱码和过度装饰约束。
- 使用的角色/环境参考图、模型、版本、种子或引用信息：如工具支持则填写。
- 封面 QA 结果：通过 / 需修正 / 必须重做；列出问题。

## 8. 内页图片与文字

每一页都必须完整给出以下字段：

### 第 X 页

- 页码：
- 本页中心意思：只写一个动作、情绪或发现。
- 本页故事文字：短、口语化、有节奏，适合朗读。
- 汉字数：不含标点。
- 朗读检查：是否顺口；是否存在成人腔、歧义或过度抽象。
- 情绪变化：本页开始时 → 本页结束时。
- 儿童主体性：孩子在本页拥有何种选择、表达或主动动作；不适用时写“无额外推动”。
- 成人/同伴支持：仅在本页出现支持行为时填写，写具体动作和语言。
- 文字与画面分工：文字说什么，画面额外补充什么。
- 页间推进：相比上一页新增什么，并为下一页留下什么期待。
- 视觉新增量：景别、动作、视角、关系、环境状态或关键细节中至少一项明显变化。
- 出场角色锁定：列出角色 ID、人数、服装、鞋袜和标志物。
- 道具状态：列出连续道具 ID、位置、持握手和本页变化。
- 空间连续性校验：起点、目标、路径轴线、固定地标、角色位置、视线和镜头方向如何承接上一页。
- 镜头切换理由：若使用反打、俯视、主观镜头或特写，说明其情绪/叙事目的以及保留的空间锚点。
- 景别：远景 / 远中景 / 中景 / 近景 / 特写。
- 视角与机位：平视 / 轻微俯视 / 轻微仰视 / 主观视角 / 反打等。
- 画面构图：前景、中景、背景、主体位置、行动方向和视觉重心。
- 人物调度与动作真实性：重心、接触、遮挡、手脚和道具关系。
- 画面细节与规整度：必须保留什么，删减什么；不为“丰富”堆叠装饰。
- 色彩设计：主色、辅助色、强调色与本页情绪的关系。
- 光线设计：方向、强弱和情绪功能；承接上一页。
- 文字融入方案：位置、占比、对齐、预计行数、安全边距；必须是后期文本层。
- 图片比例与像素尺寸：与全书锁定值一致。
- 无字插画图像提示词：包含角色 ID、固定外观、动作、镜头、环境锚点、光线、色板和自然文字空区；不得要求生成中文。
- 负面提示词：安全 + 解剖 + 一致性 + 乱码 + 场景特定约束。
- 使用的参考图、模型、版本、种子或角色引用信息：如可用则填写。
- 无字图片路径：
- 排版完成图片路径：
- 单页 QA：安全 / 解剖 / 动作 / 角色 / 道具 / 空间 / 色彩 / 文字区 / 乱码检查。
- 单页状态：通过 / P1 修正后复检 / P0 必须重做。

补充要求：

- 人物数量由本页中心意思决定，不默认每页多人；允许单人、双人、局部特写和群像。
- 正文通常不超过 3 行，不遮挡脸、眼神、手、脚、关键道具和行动路径。
- 不允许图像模型生成书中文字；所有中文由后期排版加入。
- 连续两页不能只是同一构图的轻微动作变化，除非明确用于有意义的重复节奏。

## 9. 全书质量审计

### 9.1 故事与文字 QA

- 单一核心主题：通过 / 不通过。
- 年龄适配和字数：通过 / 不通过。
- 儿童视角：通过 / 不通过。
- 情绪接纳与选择权：通过 / 不通过。
- 是否存在催促、比较、羞辱、条件式奖励或说教：无 / 有，列出并改写。
- 重复结构是否每次推进：通过 / 不通过。
- 出声朗读检查：通过 / 不通过；列出修改。
- 结尾是否允许慢慢来并避免价值评判：通过 / 不通过。

### 9.2 跨页连续性 QA

- 角色脸型、发型、服装、鞋袜和身高关系：通过 / 不通过。
- 固定配角人数与身份：通过 / 不通过。
- 道具颜色、大小、位置和持握状态：通过 / 不通过。
- 门窗、舞台、地毯、道路和家具位置：通过 / 不通过。
- 镜头轴线、行动方向和光源：通过 / 不通过。
- 色板、线条、颗粒和人物比例：通过 / 不通过。
- 画幅、像素尺寸和页码命名：通过 / 不通过。

### 9.3 AI 瑕疵审计

- 多指、少指、粘连手、反向手掌、多肢、缺肢。
- 眼睛、耳朵、牙齿、头发和衣领异常。
- 人物漂浮、脚不着地、重心不稳、物体未被真实握住。
- 重复人物、无故换装、角色变脸、道具复制或消失。
- 透视、遮挡、阴影、门窗和家具结构异常。
- 乱码、伪中文、假字母和模型生成文字。
- 同款微笑、塑料皮肤、过量纹理、随机装饰和模板化构图。

## 10. 给家长的文字

- 这个故事想让孩子感受到什么：说明孩子可感受到的生活经验和情绪，不写口号。
- 共读提醒：明确不要用故事催孩子表演、比较或证明勇敢。
- 亲子互动：给出 2-4 种可朗读、可提问、可模仿动作或可延展到生活的方式。
