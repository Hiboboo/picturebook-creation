## 3. 儿童发展与情绪安全策略

- 孩子可能感受到的情绪：好奇、疑惑、短暂停住、放松、惊喜。
- 故事如何接纳这种情绪：雷声出现时允许小松鼠停在树洞口看一看，不要求立刻继续。
- 孩子拥有的选择：观察、暂停、靠近树洞、慢慢听、慢慢看、伸出小爪试一试。
- 成人如何提供支持：本故事无成人推动；环境本身提供安全位置，树洞口可退回。
- 故事中的最小可行尝试：小松鼠停一停，再听一听雨声和风声。
- 结尾如何避免“必须克服”：结尾只是发现彩虹，不把不怕雷声当作胜利。
- 奖励、掌声或贴纸的处理：不出现奖励、掌声或贴纸。
- 禁用表达：“不要怕”“你要勇敢”“快出去看”“胆小”“真棒，所以不怕了”。

## 4. 故事设定卡

- 背景：雨前到雨后的林边小路，左侧有一棵带树洞的大树，右上方通向开阔草坡。
- 儿童视角：孩子能从叶子动、草弯、云靠近、雨点落和天边亮光理解天气变化。
- 故事弧线：生活入口 → 愿望/好奇 → 犹豫信号 → 安全支架 → 一个小动作 → 真实回应 → 温暖收束。
- 重复结构：“风在哪里？”和“这里有一点点风。”每次由不同天气线索回答。
- 页间推进：叶子动 → 追风 → 云靠近 → 雨点落 → 雷声响 → 雨变小 → 云散开 → 彩虹出现。
- 温暖结尾：彩虹出现时，小松鼠不是“战胜”天气，而是看见风和雨留下的温柔线索。
- 故事禁改项：主角为小松鼠；雷声不恐怖化；结尾是雨后彩虹；不加入成人评价或奖励。
