# 角色圣经

## C01-小松鼠

- 名称与身份：小松鼠，幼儿感主角
- 年龄感 / 物种：2-5 岁幼儿友好气质的小松鼠
- 身高与头身比例：约 2.5 头身，小体型，蓬松大尾巴约为身体高度的 1.2 倍
- 脸型与核心五官：圆脸，黑亮小眼，短鼻，小嘴，奶油色脸颊
- 发型与发色：无发型；暖栗棕短毛；深棕尾尖
- 上衣：无
- 下装：无
- 鞋袜：无
- 固定配色：暖栗棕身体，奶油色脸颊和肚皮，深棕尾尖
- 标志性配件：D01-小绿围巾
- 常见表情范围：
  - 平静：嘴角自然，眼睛放松
  - 好奇：头微歪，眼睛看向动的叶子
  - 专注：身体前倾，眼睛看向线索
  - 短暂停住：身体靠近树洞，嘴巴轻轻合上
  - 放松：肩背和尾巴放松
  - 开心：轻微微笑，不露夸张牙齿
- 禁改项：物种、圆脸、黑亮小眼、奶油色脸颊和肚皮、深棕尾尖、小绿围巾
- 可变项：动作、朝向、景别、雨点附着、围巾飘动方向
- 角色参考图路径或状态：`outputs/picturebooks/dongwu-yu-ziran-kepu/tianqi-xiao-songshu-zhui-zhe-feng-kan-caihong/bible/character-reference-sheet.png`

## D01-小绿围巾

- 归属角色：C01-小松鼠
- 颜色：柔和草绿色
- 尺寸：短围巾，两端不超过身体宽度的一半
- 图案：无图案
- 初始状态：围在脖子上，风起时轻轻向右飘
- 每页状态：
  - 第 1 页：向右轻飘
  - 第 2 页：向右后方飘
  - 第 3 页：贴近身体
  - 第 4 页：边缘可见，有少量雨点
  - 第 5 页：贴近身体
  - 第 6 页：重新轻轻向右飘
  - 第 7 页：带轻微水痕，向右后方飘
  - 第 8 页：轻轻向右飘
- 禁改项：不得变色、变长、变成披风、消失或出现在地上

## D02-树洞口

- 归属：环境安全锚点
- 位置：左侧大树根部
- 外观：圆形深棕树洞，边缘有少量苔藓，洞口尺寸约为小松鼠身体高度的 1.1 倍
- 叙事功能：第 5-6 页提供可停一停的位置
- 每页状态：
  - 第 1 页：背景可见
  - 第 2 页：左后方远处可辨
  - 第 3 页：由树冠或树干边缘延续
  - 第 4 页：不出现
  - 第 5 页：近景主要锚点
  - 第 6 页：反打锚点
  - 第 7 页：远左边缘可辨
  - 第 8 页：不出现
- 禁改项：不得移动到右侧，不得变成房门、洞穴或建筑入口
