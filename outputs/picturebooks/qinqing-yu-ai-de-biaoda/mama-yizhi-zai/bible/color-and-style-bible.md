# 《妈妈一直在》色板与风格圣经

## 画面风格

- 媒介：温暖水彩家庭感。
- 线条：细软描边，边缘轻微晕染。
- 纹理：轻纸感颗粒，避免塑料皮肤和过量高光。
- 人物比例：幼儿圆润比例，成人温和自然，表情有层次。
- 细节密度：适中偏简洁，保留叙事道具，删除无关装饰。
- 构图色彩补充：以第二版整体色调为基准，保持温暖水彩和自然家庭光；只对室内绿植叶片做轻微增绿，不做全局偏冷、偏紫或高对比处理。

## 色板

| 类型 | 色彩 | 用途 |
|---|---|---|
| 主色 | 米杏 | 墙面、整体家庭温度 |
| 主色 | 暖木色 | 地板、家具、积木 |
| 主色 | 鼠尾草绿 | 妈妈围裙、稳定连接线索 |
| 辅助色 | 奶油白 | 妈妈上衣、点心盘、柔光 |
| 辅助色 | 浅蓝 | 小米背带裤、空间清爽感 |
| 辅助色 | 暖黄色 | 小米上衣、孩子的温暖中心 |
| 强调色 | 柔和红 | 小米小鞋、小车轮，少量识别 |
| 强调色 | 柔和蓝 | D02 小车，行动线索 |

## 自然鲜艳控制

- 可以增强：室内绿植叶片。
- 增强方式：仅在叶片区域轻微提高绿色感，保持第二版整体暖色调、原有窗光、木地板、人物肤色、围裙和道具色彩。
- 不可增强成：糖果色、荧光色、随机彩虹色、大面积高饱和红黄蓝。
- 每页仍需保留低细节文字区，不能为了鲜艳而填满背景。

## 禁用倾向

- 随机彩虹色。
- 过度高饱和。
- 糖果色或霓虹色。
- 阴森低照度。
- 单一灰暗室内。
- 过量星星、花朵、气泡或无关墙画。

## 版式锁定

- 画幅：3:4 竖版。
- 像素尺寸：`1536x2048`。
- 安全边距：文字、脸部、手部、关键道具距离边缘至少 6%。
- 正文：每页 1-2 行为主，最多不超过 3 行。
- 封面：书名第一视觉层级，署名固定 `图/文：小葫芦`。
- 文字区：使用小面积自然低细节生活背景，不使用大面积空白墙、固定卡片或不透明大背板。
- 图像阶段：无字插画，禁止模型生成中文、英文、标签、墙面可读文字。
