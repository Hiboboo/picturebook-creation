## 3. 儿童发展与情绪安全策略

- 孩子可能感受到的情绪：专注、寻找、不确定、听见回应后的安心、期待、放松。
- 故事如何接纳这种情绪：小米可以抬头找妈妈、停在门口等一等，也可以继续推小车；寻找不是麻烦，等待不是考试。
- 孩子拥有的选择：观察、轻声呼唤、靠近厨房门口、抱着或推着熟悉玩具、暂停、继续玩一小会儿。
- 成人如何提供支持：妈妈提前说去厨房；从厨房回应“我在”“我听见你啦”；回来后接纳孩子靠一靠，不要求孩子证明自己没害怕。
- 故事中的最小可行尝试：小米把小车开到厨房门口，等一小会儿。
- 结尾如何避免“必须克服”：结尾是孩子感到连接和安心，不把“自己等住了”写成被表扬或换取点心的条件。
- 奖励、掌声或贴纸的处理：不出现奖励、掌声或贴纸；点心是生活事件，不是行为奖赏。
- 禁用表达：“别哭”“你要勇敢”“妈妈不要你了”“真乖，所以妈妈回来了”“妈妈忙，你不要打扰”“等好了才有点心”。

## 4. 故事设定卡

- 背景：温暖午后家中，左侧客厅、右侧半开放厨房，妈妈去厨房准备点心。
- 儿童视角：故事来自孩子能理解的身体经验和家庭线索：声音、围裙、小车、香味、妈妈回到身边。
- 故事弧线：生活入口 → 妈妈短暂离开视线 → 孩子寻找 → 声音回应 → 熟悉物件 → 一个小动作 → 真实回应 → 香味线索 → 妈妈回来 → 温暖收束。
- 重复结构：小米轻声寻找“妈妈？”；妈妈用“我在”“我听见你啦”稳定回应。
- 页间推进：搭积木 → 妈妈去厨房 → 孩子抬头找 → 听见厨房声 → 看见围裙 → 开小车到门口 → 听见回应 → 闻到香味 → 妈妈回来 → 靠一靠。
- 温暖结尾：小米主动靠一靠，说出“妈妈一直在”，让孩子感到被理解，而不是被教育。
- 故事禁改项：妈妈的爱不以表现为条件；厨房门口是安全边界；不写强迫拥抱；不写妈妈消失或惩罚性离开。
