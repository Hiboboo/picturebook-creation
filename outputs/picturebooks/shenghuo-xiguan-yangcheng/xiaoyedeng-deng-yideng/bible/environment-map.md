# 《小夜灯，等一等》环境地图

## 核心场景

- 场景：小葫芦的卧室和右侧走廊门口。
- 时间：安静夜晚。
- 天气：普通晴夜或不强调天气；不出现雷暴、闪电或强风。
- 空间基准：床在画面左侧，房门在画面右侧，窗边椅子在后侧，床头柜和小夜灯在床头。

## 平面关系

```text
后侧墙面 / 窗
┌──────────────────────┐
│ 窗边椅 + 小外套(D03)  │
│                      │
│ 小床 + 枕头 + P01     │   右侧房门 -> 走廊暖光(D02)
│ 床头柜 + 小夜灯(D01)  │
│ 床边小地毯            │
└──────────────────────┘
```

## 行动路线

- 第 1-2 页：小葫芦在床边，妈妈在床边。
- 第 3-5 页：视线从床边转向墙上影子和窗边椅背的小外套。
- 第 6 页：小葫芦回到枕头和小兔。
- 第 7 页：小葫芦看向右侧房门，妈妈把门留一条缝。
- 第 8 页：妈妈从床边退到门口坐下。
- 第 9-10 页：小葫芦在床上练习“等一等”，妈妈在门口可回应。

## 光源

- D01-小夜灯：床头柜暖琥珀光，贯穿第 2-10 页。
- D02-门缝光：右侧走廊暖黄光，第 7-10 页出现。
- 窗外夜色：灰蓝弱光，只作背景，不制造恐怖氛围。

## 固定地标

- 左侧小床。
- 床头柜和圆形小夜灯。
- 右侧房门与门框。
- 窗边小椅子和搭着的小外套。
- 床边小地毯。
- 矮书架或一两本睡前书。

## 禁改项

- 房门不得被锁上。
- 妈妈不得离开到不可回应的位置。
- 影子不得画成怪物、尖牙、爪子、额外人物或拟人恐怖形象。
- 床、门、窗边椅的左右关系不得跨页翻转。
- 卧室不得变成陌生、空旷、阴森空间。
