# 《小夜灯，等一等》色板与风格圣经

## 媒介

- 温柔水彩底色。
- 彩铅细线勾边。
- 轻纸感颗粒。
- 低对比夜色，不使用恐怖片式强明暗。

## 人物风格

- 幼儿圆润，约 3.5 头身。
- 表情细腻，不全程露齿笑。
- 成人保持真实温和，坐低或蹲低时与孩子视线接近。

## 色板

| 类型 | 颜色 | 用途 |
|---|---|---|
| 主色 | 暖琥珀 | 小夜灯、门缝光、安全感 |
| 主色 | 灰蓝 | 夜晚卧室背景 |
| 主色 | 奶油白 | 床品、妈妈上衣、柔和留白 |
| 辅助色 | 浅木色 | 床头柜、地板、椅子 |
| 辅助色 | 柔和棕 | 妈妈裤子、头发 |
| 强调色 | 淡黄色 | 小葫芦星星睡衣 |
| 强调色 | 浅蓝 | 睡衣星星、小兔耳朵 |

## 禁用色彩倾向

- 大面积纯黑。
- 阴森紫黑或高对比恐怖光。
- 随机彩虹色。
- 高饱和霓虹色。
- 过度闪亮塑料质感。

## 版式规则

- 全书固定 `3:4` 竖版，`1536x2048`。
- 正文后期可编辑文本层加入，图像模型不得生成中文或假文字。
- 文字区占画面约 `12%-20%`，使用自然低细节区域。
- 文字不得遮挡脸、手、眼神方向、小兔、小夜灯、门缝光。
- 封面书名最多 2 行；署名固定 `图/文：小葫芦`。
